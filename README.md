# Calculadora Básica
Aplicación web para realizar operaciones matemáticas básicas.

## Versiones del sistema
- **v1.0.0**: Versión inicial, solo suma y resta
- **v1.1.0**: Agregadas multiplicación y división
- **v1.2.0**: Agregadas validaciones y control de errores

## Tecnologías utilizadas
- **HTML5**: Estructura e interfaz de usuario
- **JavaScript**: Lógica de operaciones y validaciones

## Herramienta de desarrollo
- **Visual Studio Code**: Editor de código utilizado para escribir y guardar los archivos

## Funcionalidades
✅ Suma, resta, multiplicación y división
✅ Validación de datos ingresados
✅ Protección contra división entre cero
✅ Mensajes claros de error
